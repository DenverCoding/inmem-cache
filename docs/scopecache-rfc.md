# scopecache — Core RFC

> **Status: in progress.**
>
> This document will become the canonical RFC for the post-v0.7.17
> `package scopecache` core. Until it lands, the closest reference is
> [scopecache-rfc-old.md](scopecache-rfc-old.md) — the pre-strip RFC,
> kept as a snapshot. Note that the old RFC's §6.3, §6.4, §13.17, and
> §13.19-§13.23 describe endpoints that have left core; everything
> else still applies to the current core verbatim.
>
> A separate RFC will follow per add-on (`/guarded`, `/admin`,
> `/inbox`, `/multi_call`, `/delete_scope_candidates`) once those
> sub-packages land.
